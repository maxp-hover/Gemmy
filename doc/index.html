Gemmy gem

This is a general purpose gem which adds functionality to the ruby language and defines some system tasks, such as generating a gem.

It is on RubyGems. Because there is an existing gem named gemmy, the name passed to gem install is gemmyrb. But require 'gemmy' is still used.

RubyGems hosts the YARD documentation at www.rubydoc.info/gems/gemmyrb, but it's only updated when I push a new gem version. So the version on Github's master branch might contain undocumented functionality. I'll try to keep the RubyGems version up-to-date though.
